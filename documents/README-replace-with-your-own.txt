------------------------------------------------------------------
HOW TO USE YOUR OWN DOCUMENT INSTEAD
------------------------------------------------------------------
You can replace these placeholder files with your OWN document at
any time. Just keep the SAME file names so the download buttons
keep working:

  1. Create your document (in Word, Google Docs, etc.).
  2. Save / export it as:
        - PDF  -> name it  Business-Hub-Application.pdf
        - Word -> name it  Business-Hub-Application-Form.docx
  3. Copy both files into this "documents" folder, replacing the
     existing ones.
  4. Re-deploy the site (push to GitHub, Vercel updates itself).
